# 🏥 Health Care Risk Pipeline

A complete end-to-end **Machine Learning Pipeline** project that predicts health care risk levels using patient-related features. This project includes data preprocessing, model training, evaluation, and deployment via **Streamlit**.

## ⚙️ Installation & Setup

### 1) Clone the repository

```bash
git clone https://github.com/Furqan-Ahmed2006/Health-Care-Risk-Pipeline.git
cd Health-Care-Risk-Pipeline
```

### 2) Create virtual environment (recommended)

```bash
python -m venv venv
```

### 3) Activate environment

- **Windows (PowerShell):**
```bash
venv\Scripts\Activate.ps1
```
- **Windows (CMD):**
```bash
venv\Scripts\activate
```
- **macOS/Linux:**
```bash
source venv/bin/activate
```

### 4) Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

### Run training pipeline (if applicable)

```bash
python main.py
```

### Run Streamlit app

streamlit run app.py
